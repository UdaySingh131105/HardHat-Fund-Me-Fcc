# Sample Hardhat Project

This project was created by Uday Singh and it demonstrates a basic Hardhat Fund and Withdrawal functions. It comes with a sample contract, a test for that contract, and a script that deploys that contract.

Try running some of the following tasks:

```shell
npx hardhat help
npx hardhat test
REPORT_GAS=true npx hardhat test
npx hardhat node
npx hardhat run scripts/deploy.js
```
